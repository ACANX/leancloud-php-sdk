LeanCloud PHP SDK
====

_Under development_

Hacking
----

* Clone the SDK
* Install [composer](https://getcomposer.org)
* Run `composer install` to vendor dependencies
* Install php5-curl (on ubuntu)
* Setup app credential in env variables:

```
export LC_APP_ID=...
export LC_APP_KEY=...
export LC_APP_MASTER_KEY=...
```

* Run `make test` to run test
